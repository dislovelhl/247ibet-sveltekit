# 03 目标用户与使用场景

## 用户画像

- 想比较在线赌场的加拿大用户。
- 需要 NHL、NBA、UFC 或滚球说明的体育博彩用户。
- 关心 Interac 存款与提现流程的支付导向用户。
- 想确认省份规则的安大略或阿尔伯塔读者。
- 想复核奖金条款、提现时效或安全玩法的回访读者。
- 负责更新或推广页面的内部编辑 / 市场人员。

## 用户需求

- 页面开头就给出清晰答案。
- 能用很少的时间比较选项。
- 确信页面是当前且可信的。
- 看完信息后知道下一步做什么。

## 用户痛点

- 各省规则互相不一致。
- 奖金条款难懂。
- 提现时效和验证步骤不清楚。
- 页面像促销页，但没有解释权衡。
- 占位页看起来已发布，但实际上并不能帮助用户。

## 主要用户路径

| 用户类型 | 需求 | 路径 | 关键页面 | 成功结果 |
| --- | --- | --- | --- | --- |
| 赌场比较用户 | 找到更安全的游戏地点 | 首页 / 中心页 → 赌场比较页 → 游戏 / 奖金 / 支付详情 | `/`, `/casino`, `/best-online-casinos-canada`, `/casino-bonuses-canada`, `/interac-casino-canada` | 用户知道该信任哪一页，以及下一步该做什么 |
| 体育博彩用户 | 比较体育博彩平台与赛事实力 | 首页 / 体育博彩中心页 → 单项运动页面 → 工具或指南 | `/sportsbook`, `/best-sports-betting-sites-canada`, `/sportsbook/nhl`, `/tools/odds-calculator` | 用户可以比较市场并理解投注类型 |
| 支付导向用户 | 理解 Interac 存款与提现 | 支付页 → 存/提指南 → 提现时效说明 | `/interac`, `/deposit`, `/interac/withdraw`, `/fast-payouts` | 用户理解时效、限额和验证步骤 |
| 省份查询用户 | 确认本地可用性与法律背景 | 省份中心页 → 法律指南 → 支付或支持页 | `/ontario`, `/alberta`, `/legal-online-gambling-canada`, `/gambling-age-canada` | 用户理解省份差异 |
| 奖金查询用户 | 判断优惠值不值得领 | 奖金页 → 奖金条款 → 相关指南 | `/casino-bonuses-canada`, `/sportsbook-bonuses-canada`, `/bonus-terms`, `/guides/wagering-requirements-explained` | 用户在存款前理解权衡 |
| 寻求支持用户 | 找到帮助或政策信息 | FAQ / 政策 / 联系页 | `/faq`, `/contact`, `/security`, `/sources`, `/responsible-gambling` | 用户快速找到答案或支持渠道 |

## 决策因素

- 是否适用于所在省份。
- 是否支持 Interac。
- 提现速度。
- 奖金限制。
- 游戏或市场深度。
- 信任与政策信号。
- 移动端体验。

## 异议或信任障碍

- “我所在的地方合法吗？”
- “提现真的有那么快吗？”
- “奖金细则是什么？”
- “我提现吗前要不要先验证身份？”
- “这页是当前状态，还是营销文案？”
