# 11 QA 验收清单

## 页面渲染

- [ ] 首页可以正常渲染，没有控制台错误。
- [ ] 核心中心页能在桌面和移动端正常渲染。
- [ ] 重定向页面能落到正确目标。
- [ ] 动态壳页面要么显示清晰占位状态，要么显示真实内容。

## 导航

- [ ] 主导航链接都指向正确路由。
- [ ] 页脚链接都指向正确路由。
- [ ] 深层页面的面包屑可用。
- [ ] 可见导航里没有指向缺失页面的链接。

## 移动端适配

- [ ] 页面没有横向溢出。
- [ ] Sticky CTA 没有遮挡关键内容。
- [ ] 表格在小屏上仍然可用。
- [ ] 点击区域足够大。

## 表单

- [ ] 表单有标签、帮助文字和校验信息。
- [ ] 错误状态容易理解。
- [ ] 成功状态可见。
- [ ] 校验失败后不会丢失已输入内容。

## 链接

- [ ] 内链可用。
- [ ] 外链在需要时带有安全属性。
- [ ] 公开导航里没有死链。
- [ ] canonical 目标一致。

## SEO 元数据

- [ ] 每个可索引页面都有唯一标题。
- [ ] 每个可索引页面都有有用的 meta description。
- [ ] canonical 正确。
- [ ] 不需要索引的页面已设置 noindex。
- [ ] 结构化数据与可见内容一致。

## 无障碍

- [ ] Skip link 可用。
- [ ] 标题层级正确。
- [ ] 焦点态清楚可见。
- [ ] 色彩对比可读。
- [ ] 手风琴和交互控件支持键盘操作。

## 性能

- [ ] 页面在移动端的加载速度可接受。
- [ ] 图片没有明显过重。
- [ ] 加载期间没有明显布局抖动。
- [ ] 即使图片还没全部加载，关键页面也能先用。

## 内容准确性

- [ ] 法律、年龄、提现、奖金和支持信息是当前的。
- [ ] 所有来源驱动的说法都经过核实。
- [ ] 占位页有明确标识。
- [ ] 任何“待确认”都是有意保留的，而不是遗漏。

## 法律 / 合规措辞

- [ ] 没有未经证明的许可声明。
- [ ] 奖金文案与页面类型匹配。
- [ ] 责任博彩链接容易找到。
- [ ] 高意图页面有合适的披露措辞。

## 分析 / 追踪

- [ ] 分析脚本按预期加载。
- [ ] 关键点击路径已被追踪。
- [ ] workflow / audit 报告仍然指向正确位置。
- [ ] 没有遗漏上线事件。

## 上线阻塞

- [ ] 没有坏掉的 canonical 路径。
- [ ] 没有关键路由缺失。
- [ ] 没有占位壳页面被误当成已完成公开内容。
- [ ] 利益相关者已经对最终文案签字。
